# 回显某些框架和场景下的反序列化利用 

## 0x01 Introduce

1. Request/Response回显
2. 支持win和linux payload
2. base64编码

## 0x02 Support
1. ApereoCAS 4.1.X 默认密钥
2. Shiro spring 环境下回显
3. Liferay protal 7.0 以上

## 0x02 Buiding

```mvn clean package -DskipTests```
